# Interview History Entry Format

Each practice session is appended to `<things_path>/targets/interview-history.md` using this format:

```markdown
### <date> â€” <category>

**Question:** <the interview question>
**Assessment:** <Strong | Good | Needs Work>
**Key feedback:** <1-line summary of main feedback>
**Gaps identified:** <skills or areas to work on>
**Arsenal entries referenced:**
- [<log title>](../logs/<filename>)
- [<log title>](../logs/<filename>)
```

## Fields

| Field | Description |
|-------|-------------|
| `date` | Date of practice session (YYYY-MM-DD) |
| `category` | Question category: behavioral, technical, leadership, situational |
| `Question` | The exact question asked |
| `Assessment` | Overall rating of the answer |
| `Key feedback` | Most important takeaway from the feedback |
| `Gaps identified` | Skills or areas that need more evidence or practice |
| `Arsenal entries referenced` | Links to logged accomplishments used in coaching |

## Assessment Criteria

| Rating | Meaning |
|--------|---------|
| **Strong** | Specific, well-structured, demonstrated impact, owned contribution |
| **Good** | Addressed the question with some specifics, room for stronger evidence |
| **Needs Work** | Vague, lacked structure, missed impact, or deflected credit |
